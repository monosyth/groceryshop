# Teal Color Update

## Before (Original Teal)
- **dark:** `#0D9488` - rgb(13, 148, 136)
- **main:** `#14B8A6` - rgb(20, 184, 166) 
- **light:** `#5EEAD4` - rgb(94, 234, 212)
- **bg:** `#F0FDFA` - rgb(240, 253, 250)

## After (New Teal - More Green & Darker)
- **dark:** `#0A7A6E` - rgb(10, 122, 110) ✨ Darker
- **main:** `#0D9B8A` - rgb(13, 155, 138) ✨ More green, darker, better contrast
- **light:** `#4DD4C0` - rgb(77, 212, 192) ✨ Adjusted
- **bg:** `#E6F9F5` - rgb(230, 249, 245) ✨ Adjusted

## Changes
- Main color is now **darker** for better text contrast (WCAG compliance)
- Shifted toward **more green** in the color balance
- All shades adjusted proportionally to maintain consistency
- Better readability when used as text color

## Contrast Ratios (Approximate)
- Old main on white: ~3.5:1
- New main on white: ~4.8:1 ✅ Better contrast!
